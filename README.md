# styleSphere-app-mobile
App Android para catalogar roupas, montar combinações e compartilhar looks
Style Sphere

Let your Style Speak — Seu armário digital, suas combinações, do seu jeito.

Projeto desenvolvido para a disciplina de Desenvolvimento Mobile do curso de Análise e Desenvolvimento de Sistemas do Instituto Federal de Pernambuco (IFPE).

Sobre o projeto

O Style Sphere é um aplicativo Android nativo que permite ao usuário catalogar digitalmente suas peças de roupa e montar combinações de outfit de forma rápida e prática — sem precisar vasculhar o armário físico ou tentar lembrar o que combina com o quê.
Além de otimizar o processo diário de se vestir, o app tem potencial como ferramenta de acessibilidade para pessoas com dificuldades em tomar esse tipo de decisão ou gerenciar seu tempo no dia a dia.

Funcionalidades

Catálogo de Roupas

Cadastro de peças por foto
Cada peça recebe tags personalizadas pelo usuário (ex.: calça, azul, formal)
Busca e filtragem de peças pelas tags

Roleta de Combinações

Geração automática de outfits a partir de um template (ex.: blusa + calça + sapato)
Possibilidade de fixar uma peça e continuar girando as demais
Filtro por tag para restringir as sugestões (ex.: apenas roupas formais)

Combinações Salvas

Salvar outfits favoritos em categorias/pastas
Navegar facilmente pelas combinações salvas

Fórum / Rede Social

Compartilhar peças e combinações com outros usuários
Seguir usuários cujo estilo você admira


Tecnologias

Linguagem: Kotlin
Plataforma: Android (nativo)
IDE: Android Studio
